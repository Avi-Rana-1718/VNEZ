Getting started
VNEZ or Visual Novels made eazy is basically just a bunch of pre-coded files that save you from the hassle of researching, learning and writing everything yourself and saves your time! It is pretty useful for people who don't know how to code. VNEZ currently has the following features

  • No need to code the logic or the styling (Editable), just enter the dialogs, background images, character assets and other stuff.
  • Features like Fullscreen and Menu included
Installation
Download the three, they should be kept in the same folder. If the script or styling doesn't work, check whether all three of them are kept in the same folder.


Editing
We suggest that you shouldn't edit the HTML file and just stick to VNCSS and VNScript.

VNCSS contains all the styling for the Visual Novel and VNScript will be the document that shall contain all the dialogs and other assets.

VNScript
VNScript is the place where are the dialogs, background images, character assets are accessed from. Here's a quick peek on how the script shall look.


data = '[{"dialog" : "Text", "bg" : "bg.png", "c1" : "none", "c2" : "none", "ltxt" : "...", "name" : "Legend Of Love!", "author" : "AviRana"},{"dialog" : "Ohayo, Senpai!", "bg" : "bg.png", "c1" : "none", "c2" : "boy_normal.png"},{"dialog" : "How are you?", "bg" : "bg_sky.png", "c1" : "none", "c2" : "girl.png"}]';

Let me explain how to write the above code. Everything must be written between data = '[ ]'; All the parameters should be specified in { }. These brackets must be written inside data = '[ ]'; like this data = '[{ }]'; . Everything inside one { } brackets get executed at once. The next event must be specified like this data = '[{...},{...}]';
