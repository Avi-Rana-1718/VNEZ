Getting started
VNEZ or Visual Novels made eazy is basically just a bunch of pre-coded files that save you from the hasle of researching, learning and writing everything yourselves and saves your time! It is pretty useful for people who don't know how to code. VNEZ currently has the following features

  • No need to code the logic or the styling (Editable), just enter the dialogs, background images, character assets and other stuff.
  • Features like Fullscreen, Menu included
Installation
Download the three, they should be kept in the same folder to work.


Editing
We suggest that you should not edit the HTML file and just stick to VNCSS and VNScript.

VNCSS contains all the styling for the Visual Novel and VNScript will be the document that shall contain all the dialogs and other assets.
